👟 Projeto FrontEnd - Drip Store 

Projeto desenvolvido durante o programa **Geração Tech 3.0 (Digital College)**, focado em criar uma experiência de compra moderna, fluida e totalmente responsiva.

## 🚀 Tecnologias Utilizadas
- **React & Vite**: Para uma interface reativa e um ambiente de desenvolvimento de alta performance.
- **React Router**: Gerenciamento de rotas e navegação em Single Page Application (SPA).
- **CSS3 Avançado**: Uso estratégico de **Flexbox** e **Grid Layout** para designs complexos.
- **Arquitetura de Componentes**: Foco em reuso de código e manutenibilidade.

## 📋 Funcionalidades
- **Home Page Dinâmica**: Carrossel de destaques e seções de ofertas.
- **Navegação Intuitiva**: Listagem de produtos organizada e coleções em destaque.
- **Mobile First**: Layout 100% adaptável para dispositivos móveis.
